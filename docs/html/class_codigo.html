Detailed Description

Representa um identificador único de objetos no sistema formado por um código numérico de 5 dígitos.

Esta classe é utilizada para identificar entidades do sistema, como ordens ou carteiras. O valor armazenado deve conter exatamente cinco dígitos numéricos de 0 a 9. Qualquer valor fora desse padrão resulta em exceção durante a validação.
