void CPF::validarCPF ( string valor)

Valida um CPF em formato string e, se for válido, armazena‑o no objeto.

Este método chama internamente cpf_array, numerosIguais e validarNumerosCPF. Caso o CPF seja inválido, recomenda‑se lançar uma exceção ou sinalizar erro.

Parameters
cpfString com 11 dígitos (apenas números). Pontos ou traços devem ser removidos antes.
