Detailed Description

Representa uma quantia em dinheiro que é paga por um determinado produto ou serviço. Armazena um valor float representando dinheiro. Aceita e valida valores entre os limites 0.01 e 1.000.000.00 ou lança uma exceção caso o valor esteja fora dos limites.
